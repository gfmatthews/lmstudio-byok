# Changelog

All notable changes to the LM Studio BYOK Chat Provider extension will be documented in this file.

## [0.3.1] - 2026-03-15

- Makes onboarding a little easier with a UI pane to manage models and refresh


## [0.3.0] - 2026-03-14

- improved hardening of tool calling logic
- now detects and reports context window size to vsCode based on configured LMStudio context window size


## [0.2.0] - 2026-03-13

### Added

- Now correctly supports running inside a devcontainer. Yay.

## [0.1.0] - 2026-03-13

### Added

- Initial release
- Connect to local LM Studio instances via OpenAI-compatible API
- Automatic model discovery and registration
- Configurable base URL and API key
- Verbose logging option for troubleshooting
- Welcome walkthrough for first-time setup
